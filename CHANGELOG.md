## [1.0.0] - 2026-02-23

### Features
- REST API with 186 endpoints for programmatic TradingView chart control
- Symbol and resolution management
- Drawing tools (shapes, trend lines, tweet drawings)
- Alerts CRUD and management
- Watchlist management with pin/favorite support
- Pine Script editor (read/write source, save, add to chart)
- Snapshot capture with thumbnail generation
- Chart export (OHLCV + study data as CSV)
- WebSocket relay via SSE for real-time event streaming
- Passive researcher daemon for network traffic capture (JSONL)
- Optional browser launcher (Chromium via CDP)
- Notes API (chart notes, watchlist notes)
- Layout management
- Replay mode control
